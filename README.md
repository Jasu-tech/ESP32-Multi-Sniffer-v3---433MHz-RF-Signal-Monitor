## YouTube Description

ESP32 Multi-Sniffer v3 — 433MHz RF Signal Monitor

Open source tool for monitoring 433MHz radio signals using ESP32 and up to 3 CC1101 radio modules simultaneously.

What it does:
- Captures and decodes signals from car remotes, garage door openers, wireless outlets, doorbells and more
- Identifies known devices automatically (PT2262, EV1527, Nexa, Came TOP, Nice FLO...)
- Detects rolling code / undecodable signals via RSSI threshold
- Real-time web dashboard accessible from your phone (dark/light theme)
- Push notifications to your phone via ntfy when signals are detected
- Full remote control — change settings from anywhere using ntfy commands

Hardware: ESP32 DevKit V1 + CC1101 modules (1-3 radios)
Modulations: ASK/OOK, 2-FSK, GFSK, MSK, Wideband ASK
